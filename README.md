# NeuroScript Blog

This is the source code for my personal blog, built with [Astro](https://astro.build/).

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Content

Blog posts are located in `src/content/blog/`.
Neuroscript syntax highlighting is configured in `astro.config.mjs` using `neuroscript.tmLanguage.json`.
